# 문자열 형태의 수식이나 코드 실행

```python
>>> a = eval('1 + 4')
>>> a
5
```

```python
>>> a = 5
>>> exec('a = a + 4')
>>> a
9
```

```python
>>> s = '''
a = 1
a = a + 9
print a
'''
>>> exec(s)
10
```

eval 는 문자열 상태인 식을 수행한다. 한 개의 식만 받을 수 있고, 결과 값을 리턴한다.

exec 는 문자열 상태인 문을 수행한다. 여러 문장을 인자로 받을 수 있고 문자열을 실행하지만 None을 리턴한다.
